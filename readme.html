CIDENET TEST Configurar y agregar el archivo SQL y conectar su base de datos al wordpress con sus credenciales. Una vez trabajando el wordpress debemos hacer lo siguiente. Procesos para hacer la funcion de conectar el plugin personalizado con un servidor FTP: 1) Abrir un Editor de codigo (VS Code, sublime text, etc). 2) Buscar el archivo \wp-content\plugins\callftp.php 3) Editar las credenciales de su Servidor FTP en el archivo PHP en donde se declaran las primeras variables del codigo. 4) Ingresar el Shortcode call_ftp en cualquier pagina de wordpres y estara funcionando.
